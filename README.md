## Goal
To learn Rust by creating a simple number guessing game

## Setup
- Clone repo

- Run application with
```bash
  cargo run
```

## Generated docs

- To access docs use one of following
```bash
  # to generate docs
  cargo doc

  # to generate docs and open docs
  cargo doc --open
```
